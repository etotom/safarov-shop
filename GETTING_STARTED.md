# Safarov Shop - Руководство по запуску после исправлений

## 🚀 Быстрый старт

### 1. Убедитесь что зависимости установлены:
```bash
npm install
```

### 2. Запустите развития сервер:
```bash
npm run dev
```

Сайт будет доступен по адресу: **http://localhost:3000**

## 🐛 Если что-то не работает

### Featured Products все еще показывает "No Image"
1. Проверьте что файл `data/productImages.json` содержит изображения
2. Проверьте что файл `data/products.json` содержит товары с `featured: true`
3. Проверьте что пути в `productImages.json` начинаются с `/images/`
4. Убедитесь что изображения существуют в `public/images/`

### Переводы не меняются при смене языка
1. Убедитесь что компонент использует `useLanguage()` и `useTranslation()`
2. Проверьте что компонент помечен как `'use client'`
3. Проверьте что все ключи присутствуют в `lib/translations.ts` для всех 3 языков

### Товары не загружаются вообще
1. Проверьте что JSON файлы в `data/` не повреждены
2. Запустите `npm run dev` и проверьте консоль на ошибки
3. Очистите `.next` папку: `rm -rf .next` и перезапустите сервер
